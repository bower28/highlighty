Options

Highlighty is still in Beta Testing!

Some more features to come include: an interactive color picker, more custom styling, list exports, and more keyboard shortcut options.
Feel free to email any thoughts or suggestions to stephen@wustep.me.

Highlighty is open-source and you can contribute!

Notes:
- Highlighty has two modes: Manual and Auto.
- Manual: Extension button highlights phrases in tab.
- Auto: Extension button toggles auto-highlight.
- A green icon indicates Auto-Highlight is toggled on.
- A yellow icon indicates the tab was manually highlighted.
- A blue icon indicates that highlighting is off.
- If Highlighty fails to work, try refreshing or reinstalling!
