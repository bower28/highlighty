Options

Highlighty is still in Beta Testing!

Highlighty is open-source and you can contribute!


Notes:
- Highlighty has two modes: Manual and Auto.
- Manual: Extension button highlights phrases in tab.
- Auto: Extension button toggles auto-highlight. Phrases should get automatically highlighted.
- A green icon indicates Auto-Highlight is toggled on.
- A yellow icon indicates the tab was manually highlighted.
- A blue icon indicates that highlighting is off.
- A red icon indicates Auto-Highlight is blacklisted and can be manually toggled on.

- If Highlighty fails to work, try refreshing or reinstalling with the latest update!
- If you have small phrases ("an"), they may take priority over large phrases ("anteater") in your list with Partial Phrase Matching turned on.
- Highlighty might not work on every website. If you encounter any bugs or have any suggestions, please contact me!
